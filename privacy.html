1. はじめに

CommentTrans（以下「本サービス」）は、 利用者のプライバシーを尊重し、 個人情報および取得した情報を適切に取り扱います。

2. 取得する情報

本サービスは、コメント翻訳機能を提供するために、 以下の情報を取得する場合があります。

  • YouTube Liveチャットのコメント本文
  • コメント投稿者の表示名
  • コメント投稿日時
  • 利用者が指定したYouTube Live配信URL
  • Discord上で本サービスを利用するために必要な情報

3. 情報の利用目的

取得した情報は以下の目的のために利用します。

  • コメント翻訳機能の提供
  • サービスの運営および保守
  • 不具合の調査および改善
  • 不正利用の防止

4. 情報の保存について

本サービスは、翻訳処理のために必要な範囲で情報を一時的に処理します。

運営上必要な場合を除き、 YouTube Liveチャットの内容を恒久的に保存することはありません。

5. 第三者提供

本サービスは、法令に基づく場合を除き、 取得した情報を第三者へ販売または提供しません。

6. YouTube API Servicesの利用

本サービスはYouTube API Servicesを利用しています。

YouTube API Servicesを通じて取得した情報は、 コメント翻訳機能の提供のためにのみ利用されます。

利用者は、本サービスの利用にあたり、 YouTubeの利用規約およびGoogleのプライバシーポリシーが適用される場合があることを理解するものとします。

  • YouTube 利用規約
  • Google プライバシーポリシー

7. セキュリティ

本サービスは、取得した情報の漏えい、紛失、 改ざん等を防止するため、合理的な安全対策を講じます。

8. プライバシーポリシーの変更

本サービスは、必要に応じて本ポリシーを変更することがあります。

変更後の内容は、本サイトに掲載した時点で効力を生じるものとします。
