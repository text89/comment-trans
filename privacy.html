ホーム プライバシーポリシー 利用規約

1. 基本方針

CommentTrans（以下「本サービス」）は、 利用者のプライバシーを尊重し、 取得した情報を適切に取り扱います。

2. 取得する情報

本サービスは、コメント翻訳機能を提供するため、 以下の情報を取得する場合があります。

  • YouTube Liveチャットのコメント本文
  • コメント投稿者の表示名
  • コメント投稿日時
  • 利用者が指定したYouTube Live配信URL
  • Discord上でのサービス利用に必要な情報

3. 情報の利用目的

取得した情報は以下の目的で利用します。

  • コメント翻訳機能の提供
  • サービスの安定運用
  • 不具合の調査および改善
  • 不正利用の防止

4. 情報の保存について

本サービスは翻訳処理に必要な範囲で情報を利用します。

YouTube Liveチャットのコメント内容は、 サービス運営上必要な場合を除き恒久的に保存しません。

5. 第三者提供

本サービスは、 法令に基づく場合を除き、 取得した情報を第三者へ販売または提供しません。

6. YouTube API Servicesの利用

本サービスはYouTube API Servicesを利用しています。

YouTube API Servicesを通じて取得した情報は、 コメント翻訳機能の提供のためにのみ利用されます。

利用者は、本サービスの利用にあたり、 YouTube利用規約およびGoogleプライバシーポリシーが適用される場合があることを理解するものとします。

  • YouTube利用規約
  • Googleプライバシーポリシー

7. セキュリティ

本サービスは、 取得した情報の漏えい、紛失、改ざん等を防止するため、 合理的な安全対策を実施します。

8. プライバシーポリシーの変更

本サービスは、 必要に応じて本ポリシーを変更することがあります。

変更後の内容は本サイトに掲載した時点で効力を生じるものとします。

9. お問い合わせ

本ポリシーに関するお問い合わせは、 こちらのフォームaからご連絡ください。
