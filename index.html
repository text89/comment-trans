CommentTransとは

CommentTransは、YouTube Live配信者を支援するためのコメント翻訳サービスです。

配信中に投稿された外国語コメントを自動的に日本語へ翻訳し、 海外視聴者とのコミュニケーションをサポートします。

本サービスはDiscord Botとして提供されており、 Discord上から簡単に利用できます。

主な機能

  • YouTube Liveコメントの取得
  • 外国語コメントの日本語翻訳
  • Discord Botによる操作
  • YouTube Live URL指定による起動
  • 配信終了時の自動停止

利用方法

  1. DiscordサーバーでCommentTransを利用します。
  2. DiscordコマンドからYouTube LiveのURLを指定します。
  3. 翻訳処理が開始されます。
  4. 配信終了時に自動的に停止します。

利用について

CommentTransは非営利目的で運営されています。

利用をご希望の場合は、 管理者までフォームからお問い合わせください。

プライバシーについて

CommentTransは翻訳機能を提供するために、 YouTube Liveチャットのコメント情報を取得します。

取得した情報は翻訳機能の提供およびサービス運営のためにのみ利用されます。

詳細は プライバシーポリシー をご確認ください。

YouTube API Servicesの利用について

CommentTransはYouTube API Servicesを利用しています。

本サービスを利用することにより、 利用者はYouTube利用規約にも同意したものとみなされます。
